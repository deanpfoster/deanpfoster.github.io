Optimization in S-Plus and R

The key commands available in S-Plus are

  • nlminb Local minimizer for smooth nonlinear functions subject to bound-constrained parameters. That's maximization of smooth functions over a hypercube.
  • optimize Approximates a local optimum of a continuous univariate function within a given interval.
  • uniroot Approximates a root or zero of a continuous univarite function given an interval for which the function has opposite signs at the endpoints.

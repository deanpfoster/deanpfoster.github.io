Outline

  • Introduction
    • Certificate of design: blue prints
      • Try to get prints for fancy library
      • DNA is blue print of life
      • So no question that life was designed
      • Question is, was it intelligent
    • Rational vs intelligent
      • economic defintion of rational = maximizing
      • Unfortunately, there is no good definition of intelligent
      • We will take intelligence to be the ability to solve problems, but which problem does it try to solve? And how good is it at solving those problems?
      • We will assume that any designer is at least as smart as a human
      • A variety intelligence exists:
        • rational (smart and maximizing)
        • artistic (generates symetry, and pretty patterns)
        • lazy (reuse previuos designs rather than finding correct one)
        • sufficing (just good enough to get the job done)
        • humorious (works on generating a good laugh, see Douglas Adams)
        • imitative (imitates other designers or methodologies)
    • Static vs. dynamic behaviours
      • Static is built and ignored (Eg the pyramids)
      • dynamic requires continuous maintenance (eg Mininasota roads)
    • Evolution: Generate and test.
      • An algorithm not a variety of intelligence
      • a well specified process
      • But not very good at solving problems
  • Cryptography, security and DNA
    • Simple subsitution cipher
      • Using generate and tests it takes n! time to decypher it
      • Size of n is important: table of bits vs CPU time to crack
      • US governments DSA = 54 bits (check name and bits)
    • The code of life
      • Using permutation on 21 alphabets > 60 bits
      • Describe cipher

      • CJF? (or is this truely a prion?)

    • Ramifacations

      • If Great Danes and toy poddles can't physically mate, we could still tell if they were the same species by checking their DNA code
      • There is an audit trail of any species generated by evolution. These species all would have the same DNA code.

  • Conclusions
    • The standard model:
      • evolution generated all life (over a period of several billion years)
      • humans are a small step away from the animals
      • There is no intelegent design
      • predictions: see courses on evolution. Read Dawkin's / Dennet.
    • One young earth model:
      • All animals are designed by an intellegent designer less than a million years ago
      • This designer is best modelled by saying it is lazy, sufficing and imitative (i.e. non-rational)
      • need more conditions before predictions that differ from standard model can be made
    • Common physicist view: (called "The Anthromophic Principle")
      • Static design done at big bang. (i.e. no involvement since then by the designer)
      • biologists explain biology, so if they say evolution is good enough, no need to get involoved.
      • (minority position: selection bias generated intelligence.)
      • minority position predicts: we are alone in the universe.
    • Active designer (typically believed to be an ET. See David Birn's books about "Uplifting.")
      • evolution generated all animals, plants, etc except for humans.
      • Assumption: intelligence (like humans have) is difficult to evolve and hence doesn't happen without guidence
      • careful breading over several million years turned apes/chimps into humans.
      • Prediction: it will take millions of years to breed dogs to be intelligent and learn to speak.

Hints on the PDL stuff:

  • Background: Suppose you had a regression of Y = alpha + beta1 X1 + beta2 X2 + noise. But you wanted to assume that beta1 = beta2. How would you do it? The easiers way would be to construct a new varible Z = X1 + X2 and do a regression of Y on Z, namely, Y = Z + noise.
  • PDL: We want to assume that beta1, beta2, ..., beta6 land on a polynomial when plotted verse their index number. In other words, we want betai = a + b*i + c*i2 + d*i3. To do this we ahve to create the following variables:
    • Sum = X1 + X2 + X3+ X4+ X5 + X6
    • L = Linear = X1 + 2 * X2 + 3 * X3 + 4 * X4 + 5 * X5 + 6 * X6
    • Q = Quadratic = X1 + 4 * X2 + 9 * X3 + 16 * X4+ 25 * X5+ 36 * X6
    • C = Cubic = X1 + 8 * X2 + 27 * X3 + 64 * X4 + 125 * X5 + 216 * X6
    Now do a regression on Sum,L,Q and C.

    Finding out the coeficient of say X2 can be constructed by using betaSUM + 2*betaL+4*betaQ+8*betaC.

  • (PDL for part c) Now suppose that we want the coefficient of X6 to be zero. We can create the following variables:
    • Sum' = Sum - C/216
    • L' = L - C/36
    • Q' = Q - C/6
    Now a regression on Sum', L' and Q' will generate coefficients betaSum' and betaL' and betaQ'.

    From these we can figure out what the coeficients are for the unprimed variables. To do this use:

    • betaSum = betaSum'
    • betaL = betaL'
    • betaQ = betaQ
    • betaC = - betaSum'/216 - betaL'/36 - betaQ'/6
    Now we can figure out what the coefficients are for the orginal X's. With a bit of algebra, you should be able to show that the coefficient of X6 is in fact zero.

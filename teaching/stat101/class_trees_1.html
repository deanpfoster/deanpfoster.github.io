    4 million profit if successful. 2.5 million loss if not successful.

    P(success) = .6, P(+ test|success) = .8, P(+ test|failure) = .4.

    • E(profits perfect info) = 4*.6 = 2.4 million
    • Value of perfect information = 1 million.
